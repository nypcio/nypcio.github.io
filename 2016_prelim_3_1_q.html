Calculate your best lap

Kartrider is a driving game released in 2004 that is still loved by many players.

One game in Kartrider has many laps. Essentially, the player who enters the finish line first in the very last lap is considered the winner, but there are still times recorded for the best lap.
Write a program that outputs the fastest time one player drove in all of the L laps the players drove.

Input

In the first line, L, the number of laps completed by the player is given. L should be greater than or equal to 1 and less than or equal to 50. Across the next L lines, in the format of "00:00.000", the time taken for driving each individual lap is given. Minute and seconds are always between 0 and 59 inclusive, and millisecond is between 0 and 999 inclusive.

Output

Output the lap number L with the fastest time. If there are multiple laps with the fastest time, output the last lap number.

Input Example 1

       3
       03:15.085
       03:12.199
       03:09.182

Output Example 1

          3

The last lap had the fastest time.

Input Example 2

       3
       01:59.000
       02:00.000
       01:59.000

Output Example 2

          3

There were multiple laps with the same shortest time, so output the last lap number.
