Release the Squirrels!

Long long time ago, in 1996, players of the game “Land of the Wind” used to say: “Release the squirrels!”

If new squirrels were not released during hunting seasons when there were no squirrels left to hunt, players would say that message to rush Nexon to release more squirrels.

In 2038, 42 years after the release of game “Land of the Wind”, “주모 왈숙” decided to change the rules by introducing a new type of skill.

The new technology, DoD(DaramG on Demand), guaranteed that there wouldn’t be a situation where the squirrels are lacking, and in fact, it made sure that there were twice the amount of squirrels in comparison to the number of hunters on the hunting ground.

If given a game screen of size N by N and the number of squirrels in each section, write a program that determines if there needs to be more squirrels on the screen, and outputs the locations where the new squirrels should be placed if needed.

Input

First line of input will have the size of the game screen, N. N is an integer greater than or equal to 5 and less than or equal to 20.

In the next N lines of input, N characters will be given without any space in between.

The character will be one of these three options:
- D: There is a Squirrel in that area
- C: There is a player in that area
- . : The area is empty

Output

Following the same instructions as used in the input, the first line of output should be the size N.
Across the next N lines of output, the updated game screen information where each area is represented by either D, C, or . should be printed.

New squirrels still use the same letter, D, and they can be placed anywhere as long as the number of squirrels is correct.

If there is no need for new squirrels to be released, then the same game board inputted should be printed.
However, if there is a lack of squirrels, the number of squirrels in the resulting game board should be twice the number of players currently on the hunting ground.

Squirrels’ locations and players’ locations cannot overlap.
There will never be a case where new squirrels cannot be released if needed.

Input Example 1

 
       5
       .....
       .....
       .C...
       .....
       .....

Output Example 1

       5
       D....
       ..D..
       .C...
       .....
       .....

Input Example 2

       5
       DD...
       .D...
       .D...
       .DD..
       .....

Output Example 2

       5
       DD...
       .D...
       .D...
       .DD..
       .....

If there are only squirrels present on the board, then the number of players is 0, so there is no lack of squirrels.
Therefore, do not relase more squirrels.
