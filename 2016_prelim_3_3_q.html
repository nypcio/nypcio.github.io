A Clock with only an hour hand

Bazzi dropped the only clock present in the house. As a result, only the hour hand remains.

Bazzi doesn't wish to spend money on a new clock and he believes that he will be able to tell the exact time with only the hour hand by looking at the exact location of it, so he decided to write a program that will determine the exact time for him given the location of the hour hand.
Let us design a program that outputs the exact time given the exact location of the hour hand in a perfect circle clock.

Input

The x and y coordinate of the tip of the hour hand is given in integers. The maximum absolute value for both values are less than or equal to 1,000,000. If the center of the clock is (0, 0), 3'o clock direction is the positive x-axis, and the 6'o clock direction is the positive y-axis. Therefore, if the clock is exactly pointing at 12, x is 0 and y is negative.
The length of the hour hand is not given, but it's precise enough so that we can always tell the exact time even until the second.

Output

Output the most exact time possible in the format of HH:MM:SS. One digit integer will have a 0 added to the front, so HH goes from 01 to 12, MM goes from 00 to 59, and ss goes from 00 to 59 as well. One second difference from the exact time will be considered correct, so it doesn't matter if you decide to drop or up the time by 1 second.

Input Example 1

       -866025 -500000

Output Example 1

          10:00:00
