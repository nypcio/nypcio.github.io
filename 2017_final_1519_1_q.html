Create a Rock Axe

In Durango, the new mobile MMORPG, players need to either build or gather wild objects for their survival. For example, they can follow the rules below to acquire objects necessary for their survivals:
      - They can get one woodstick from nature. (energy 10 needed)
     - They can get one reed from nature. (energy 5 needed)
      - They can get one rock from nature. (energy 5 needed)
      - They can make one wooden stick from one woodstick. (energy 10 needed)
      - With one rock, one wooden stick and two reeds, they can make one rock axe. (energy 50 needed)
The total energy needed to make one rock axe is 10 + 10 + 10 + 5 + 50 = 85.
Given all the rules, write a program that tells us in what order what actions should be taken to create a rock axe.

Input

The first line of input has N, the number of rules (1 ≤ N ≤ 50).
Across the next N lines, rules for making tools are given in the form of k m1 c1 m2 c2 ... mk ck E M. This means that k (0 ≤ k ≤ 4) m1,m2, .. mk ingredients use up c1, c2 ... ck (1 ≤ c_i ≤ 10), and spend E energy (1 ≤ E ≤ 100), you can make one M. If k = 0, it means that this tool is possible to collect from nature. M is a 16-long lowercase alphabet sequence. There may be multiple rules for making the same object.
In the given M, there is always a "stoneaxe" and this represents the rock axe.

Output

Output K, the number of steps needed on the first line. This should be less than or equal to 100 but greater than or equal to 1. Across the next K lines, output the process of which rule should be taken how many times.

Input Example 1

5
0 10 branch
0 5 reed
0 5 stone
1 branch 1 10 woodstick
3 stone 1 woodstick 1 reed 2 50 stoneaxe

Output Example 1


5
2 2
1 1
4 1
3 1
5 1

Input Example 2

3
0 8 stone
1 stone 2 1 stone
1 stone 3 3 stoneaxe

Output Example 2


2
1 3
3 1

Marks Guideline

The input cases have 2 types: To get points, you need to have correct outputs for all test cases for one type of input test cases.
          - Type 1 (23 marks): There is only one rule for making one tool.
          - Type 2 (77 marks): No other restrictions other than the ones listed in the problem.
If there is no error in your ouput, and based off your output you have successfully created one rock axe, then you get half the marks. If it is the minimum amount of energy used, you will get the full amount.
