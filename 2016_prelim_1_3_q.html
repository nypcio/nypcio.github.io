Erin's Clock

In Erin, Mabinogi's world, we use a different calendar from the ones used in the real world. First, the year is separated into seven semesters. The seven semesters are:
         - Alban Eiler
         - Beltane
         - Alban Heruin
         - Lughnasadh
         - Alban Elved
         - Samhain
         - Imbolic

Each semester has 40 days and so the entire year in Erin is 280 days. One day is 24 hours and 1 hour is 60 mins.
Erin's time passes 40 times faster than time does in the real world. Therefore, one day in the real world is Erin's one semester, and one week in the real world is Erin's one year. Furthermore, Alban Eiler is Monday, and Imbolic is Sunday if corresponded to the real world days from Korean Standard Time. Meaning:
         - Monday: Alban Eiler
         - Tuesday: Beltane
         - Wednesday: Alban Heruin
         - Thursday: Lughnasadh
         - Friday: Alban Elved
         - Saturday: Samhain
         - Sunday: Imbolic
So if it is Monday in Korea, it is Alban Eiler in Erin, and if it's Samhain in Erin, it is Saturday in Korea. Given the date and time in the real world, let us create a program that determines Erin's date and time. We do not consider leap seconds nor Erin's year.

Input

First line of input has year, month, day, hour, minute, second separated by spaces which are all represented by integers. The given times are in the timeframe of 06/22/2004 and 12/31/2100.

Output

In Erin, we write: [Day], [Semester], [Hour]:[Minute].
Day is represented as for example, 3rd, 8th, 21st and the rules for this are as follows:
         - If the day's ten's digit is 1, we write the number and write "th" at the end. ex) 11th, 12th, 13th, etc.
         - If the day's ten's digit is not 1 :
               - If the one's digit is 1: we add "st". ex) 1st, 21st, etc.
               - If the one's digit is 2: we add "nd". ex) 2nd, 22nd, etc.
               - If the one's digit is 3: we add "rd". ex) 3rd, 23rd, etc.
               - If none of the above cases apply, we add "th". ex) 9th, 20th, etc.
When we are calculating hour and minute, sometimes they don't divide exactly. For such a case, take a look at the examples.

Input Example 1

          2016 1 1 17 10 30

Output Example 1

       29th, Alban Elved, 15:00

1/1/2016 is a Friday. 29th, Alvan Elved, 14:59, 29th, Alban Elved, 15:01 are both considered correct.

Input Example 2

        2016 6 1 11 59 59

Output Example 2 

        20th, Alban Heruin, 23:59

If it becomes midnight in the real world, in Erin, the entire semester changes. Therefore, 11:59:59 is just 20th, 11:59pm.
20th, Alban Heruin, 23:58, 21st, Alban Heruin, 00:00 are both considered correct.

Input Example 3

        2016 2 29 17 29 38

Output Example 3

        30th, Alban Eiler, 03:45

2016 was a leap year.
