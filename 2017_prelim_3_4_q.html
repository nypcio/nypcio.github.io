Ant and Road Tiles

An ant is trying to move to a destination on a 2-D plane. On this 2-D plane, there are road tiles patterned as shown below. The lines of the image represent the boundary of road tiles and continuous shapes are drawn. Some points are given coordinate values.

One of the top tiles are covered by dirt, so the boundaries of the tiles are blocked. In the above diagram, we can see that 3 tiles are blurred from (2, 8) to (18, 8).
For an ant, crossing the boundary borders is difficult, so we want to select a path that crosses the least amount of borders but still gets them to its destination. In the above diagram, the ant can get to the destination by crossing 4 boundaries.

Write a program that outputs the least number of boundaries the ant has to pass to get to its destination point given the ant's starting coordinate, destination coordinate, and places wehre borders are blurred out.

Input

The first line of input will have the number of borders deleted represented by N. (0 ≤ N ≤ 3000)
The next N lines will have the mid point coordinates of deleted tiles one by one. The line following these N lines, the starting coordinate of the ant will be given, followed by the destination coordination of the ant. Every coordinate should be greater than or equal to -100,000,000 and less than or equal to 100,000,000. there will not be a case where the ant's starting coordination and end coordination will lie on the border.

Output

Output the least number of borders the ant needs to cross to get to the destination.

Input Example 1

       3
        6 7
        10 7
        14 7
        2 1
        18 5

Output Example 1

        4

Input Example 1

      0
      2 1
      18 5

Output Example 1

        5

Marks Guideline

The input cases have 3 types: To get points, you need to have correct outputs for all test cases for one type of input test cases.
          - Type 1 (80 marks): All coordinates are in the range of 1 to 100.
          - Type 2 (100 marks): N = 0
          - Type 3 (170 marks): No other restrictions other than the ones listed in the problem.
