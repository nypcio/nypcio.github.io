Square

You are given a polygon on a 2-D plane. The given polygon has 6 points and is in the shape of an L. The edges of the polygon are always either horizontal or vertical and the lengths are given in the following order: Starting from the top-left point to the top-right point that edge's length is given first, and the remaining edges' lengths are given in clockwise order.

In this problem we wish to count the number of squares with edges length less than or equal to K that can be drawn inside this polygon. The squares may overlap. The squares' edges must be parallel to the edges of the polygon and the edges' lengths should always be an integer. For the image above, we can see that the number of squares we can draw with edges length 1 is 14. For edges length 2, we can draw 7 squares. Therefore, if K = 2, the answer becomes 21.

One thing that we need to be careful of is that it is possible for the 3rd length given to be 0. In that case, the given polygon is a rectangle. First, second, and fourth length given are always greater than 0.

Input

The first line of input has a, b, c, d, and K. a, b, d, K are all integers greater than 0, and c is an integer greater than or equal to 0. All values are less than or equal to 100.

Output

Output the number of squares you can draw given K.

Input Example 1

     3 2 1 2 2

Output Example 1

        21

Marks Guideline

The input cases have 4 types: To get points, you need to have correct outputs for all test cases for one type of input test cases.
          - Type 1 (10 marks): c = 0, K = 1.
          - Type 2 (10 marks): K = 1.
          - Type 3 (10 marks): c = 0.
          - Type 4 (40 marks): No other restrictions other than the ones listed in the problem.
