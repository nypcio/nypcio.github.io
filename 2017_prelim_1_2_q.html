Campfire

Mabinogi's campfire is loved by a lot of users because of its unique warmth. In town, in dungeon, wherever the users gathered to chat with each other, campfires were always there. However, if the distance between the campfires are too close to one another, there is a dnager of creating a big fire. Therefore, we need to reject a request to build a new campfire near an already built campfire. Let us create a program that receives requests for building a new campfire plus monitoring the requests.

The location of the campfire is labelled as (x, y) on a coordinate plane. The straight distance between campfires must be at least 5, and we need to reject any campfire requests that may cause the straight distance between campfires to be less than 5.

If the campfire started at time t, it goes off at time t + 300. Turning off the campfire takes priority over a new campfire request. If a campfire was started at time t = 10 at (x, y) = (1, 3), then since it turns off at t = 310, campfire at (x, y) = (2, 4) can begin at t = 310.

Input

First line of input has the number of starting campfire requests, N. N is an integer between 1 and 100,000 inclusive.
In the next N lines, t, the time each request was made will be listed with the location of campfire request (x, y). t is an integer between 1 and 10,000 inclusive and both x and y are integers greater than or equal to -3000 and less than or equal to 3000. Each information will have a space in between the time and the location.
The times for each request will be given in chronological order.
Sometimes requests can come in at the same time, in which case, we will file the requests in the order they were entered.

Output

For all N requests, output one line for each request whether the request was accepeted or not. If a campfire was built normally, print S_OK, otherwise print E_FAILED.

Input Example

        9
        0 0 0
        0 0 0 
        2 4 0 
        4 2 3 
        7 5 0 
        12 -3 -4
        299 2 3
        300 0 0
        301 5 2

Output Example

        S_OK
        E_FAILED
        E_FAILED
        E_FAILED
        S_OK
        S_OK
        E_FAILED
        S_OK
        E_FAILED
