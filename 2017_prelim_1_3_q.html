Postpositional Particle

When we are creating a game, we have set a structure for a sentence and by simply changing up the words in the structure, we modify the outputting sentence. However, in Korean we have different types of postpositional particles such as: 은, 는, 이, 가, 을, 를. If the letter right in front of a postpositional particle is a consonant, we use: 은, 이, 을, otherwise we need to use 가, 는, 를.

For instance, the two sentences below follow the same sentence structure. In between the sentences, we simply changed "스켈레톤" to "뱀파이어", but because of the different words, we notice the changed postpositional particles.
          - 스켈레톤을 쓰러뜨렸다!
          - 뱀파이어를 쓰러뜨렸다!

We can use "스켈레톤을(를) 쓰러뜨렸다!" but this makes the game seems less polished, so it's important to deal with the different cases of postpositional particles. There are more postpositional particles, but we will only deal with 은, 는, 이, 가, 을, 를.
