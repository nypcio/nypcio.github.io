Mabinogi Dual: Is this a valid Deck?

In the mobile TCG Mabinogi Dual, there are five types of elements: Gold, Mana, Light, Dark, and Nature.

The cards are divided depending on its element.
For example, the following card has a blue glass icon, and that means that this card requires the element Mana.

In order to start the game, you need to gather 12 or less cards to form a deck.
Each deck can have a mixture of max three elements.

Given the list of elements 12 or less cards require, write a program that determines whether these cards can form a valid deck.

Input

First line of input will have the size of the game screen, N. N is an integer greater than or equal to 1 and less than or equal to 12.

In the next N lines of input, each card’s required element (gold, mana, light, dark, or nature) will be given.

Output

If these cards can form a valid deck, output “valid.” Otherwise, output “invalid.”
