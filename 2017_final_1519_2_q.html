Guild

In a game, there are N users. These users have friendships with one another, so everyone has N - 1 friends. Each user has their skill score and this is an integer value. It can be a negative integer.

The above image shows the relationship of 8 users and their friendshps along with their skill scores. The users can make a guild, and a guild must have more than one user and if you are in the same guild, two users must be connected to another in more than just one friendship. The guild's score is the sum of all the user's skill scores.

In the above scenario, to create a guild with the maximum score is joining the users with skill scores 2 and 3. In this case, the guild score becomes 5. Of course, a guild with one user with a skill score 5, makes the guild score of 5.
Let's say the top user with a -4 skill score increased its skill score to 3. Then, four users can join to make a guild with a score of 9. If the bottom left user with a skill score 2 increased its skill score to a 5, then 6 users can join to make a guild with a score of 10.
Given the friendship map, beginning skill scores of each users and their possible increase in scores, write a program that outputs the way to create a guild with the highest score.

Input

The first line of input has N, the number of users. (1 ≤ N ≤ 100,000). Every user has a number labelled from 1 to N. Across the next N lines, skill scores are given for each user. The skill scores are always greater than or equal to -109 and less than or equal to 10 9. In the next N-1 lines, we are given the friendships. The next line has the number of upgrades or downgrades of skill scores, U. (0 ≤ U ≤ 100,000) The following U lines have the difference of users with the user number n, and the variable number v. v is always greater than or equal to 1 and less than or equal to 10 9.
In the given M, there is always a "stoneaxe" and this represents the rock axe.

Output

Output is done across U + 1 lines. The first line has the maximum guild score with the original information, and the following U lines have the maximum guild score depending on each U variatin.
